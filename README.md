## RMLProcessor Server

### Requirements
- Node.js
- Download and install the [RML-Mapper](http://git.mmlab.be/rml/RML-Mapper).

### Usage
- Clone this repo.
- `cd rmlp_server`
- `npm install`
- update `config.json`
- Start the server via `npm start`.
- The server is available at `http://localhost:4000` (if port is unchanged).
